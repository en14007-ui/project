# Simple Breakout (Arkanoid Clone)
Classic brick-breaker game with modern twists — built from scratch in **C++** using **Raylib**.
## Features

- Realistic paddle + ball physics with angle-based bounces
- Multiple levels with different brick layouts
- Boss fight on the final level (moving boss with health bar)
- Power-ups that fall from bricks (big paddle mode implemented)
- Coin system — special blocks drop coins when destroyed
- Lives, score, and coin counters
- Pause menu (ESC)
- Victory & Game Over screens with animated effects
- Background music + sound effects
- Clean code structure (separate files for ball, paddle, level, graphics, assets)

## Controls

- **A / ←** — Move paddle left  
- **D / →** — Move paddle right  
- **SPACE** — Start game / restart after win or lose  
- **ESC** — Pause / resume game
# Technologies

- **C++17**  
- **Raylib** (graphics, input, audio)  
- Custom level loading (char grid + health array for multi-hit bricks)  
- Resource management (load/unload textures/sounds/music)
